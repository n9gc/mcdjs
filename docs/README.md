# McdJS - 指令生成器

![banner](../packages/mcdjs/lib/banner.svg)

## 简介

McdJS 是一个能够让你用 JavaScript / TypeScript 写出我的世界指令的工具。

本工具使用《MC 指令设计》系列教程中的思想对我的世界指令进行抽象，将我的世界指令变得简单易懂，符合逻辑。
同时，你还可以享受到 JavaScript / TypeScript 生态所带来的便利，例如编辑器代码提示、持续部署、版本控制等。
